##  Project Overview
This project focuses on analyzing insurance business data using **Power BI** to gain actionable insights into sales performance, customer behavior, and policy trends.  
The dashboard helps management make data-driven decisions by presenting key metrics and trends in a clear and interactive way.

---

##  Objectives
- Analyze overall insurance sales performance  
- Understand customer and policy distribution  
- Track revenue trends over time  
- Identify high-performing insurance categories  
- Provide a clear business overview using KPIs  

---

##  Data Model & Design
- **Fact Table**: Insurance sales transactions  
- **Dimension Tables**:
  - Date  
  - Customer  
  - Policy / Product  
- **Schema Used**: Star Schema  

Relationships are created using **primary and foreign keys** to ensure accurate analysis.
